# Pitt State Book Buyback

## Motivation
The Pitt State Book Buyback tool is designed to automate the process of checking textbook buyback prices for students at Pittsburg State University. By programmatically retrieving pricing data, this project helps students quickly identify the best value for their course materials without manual searching.
